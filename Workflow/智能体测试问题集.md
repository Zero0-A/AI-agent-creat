## 1. 测试用例清单（30 条）

### A 类：信息完整用户

> **核心测试目的**：画像提取是否准确、推荐结果是否合理、备赛计划是否生成。

---

**T01 — 计算机 + 有项目 + 冲省奖**

| 字段 | 内容 |
|---|---|
| **测试类型** | 信息完整用户 |
| **用户输入** | 我是计算机科学与技术专业大二，会 Python 和一点机器学习，有一个课程设计是校园智能问答系统。暑假每周可以投入 20 小时，希望拿省奖，最好对保研有帮助。请推荐适合我的比赛。 |
| **预期路径** | 推荐路径（画像提取 → 知识库检索 → 匹配评分 → 推荐生成 → 备赛计划） |
| **预期行为** | 1. 提取出 major=计算机, grade=大二, skills=[Python,机器学习], available_time=暑假每周20小时, award_goal=省奖<br>2. 推荐至少 3 个匹配竞赛<br>3. Top 1 应为计算机/编程/智能体类竞赛<br>4. 推荐理由包含"项目适合转化参赛"（因为有课程设计）<br>5. 输出包含备赛计划（阶段划分+时间+任务+产出） |
| **通过标准** | ① 画像包含全部 5 个必填字段<br>② Top 1 为编程/软件类竞赛<br>③ 理由中提及"项目转化"或"项目基础"<br>④ 备赛计划不少于 3 个阶段<br>⑤ 无胡编的报名时间 |

---

**T02 — 机械 + 有项目 + 偏好创新创业**

| 字段 | 内容 |
|---|---|
| **测试类型** | 信息完整用户 |
| **用户输入** | 我是机械设计制造及其自动化大三学生，会 SolidWorks 和 AutoCAD，有一个无人机项目，申请了专利。平时每周能投入10小时左右，想参加创新创业类比赛，目标国奖。 |
| **预期路径** | 推荐路径 |
| **预期行为** | 1. 专业匹配到机械类竞赛<br>2. 有项目+有专利 → 适合创新创业类竞赛转化<br>3. 推荐应包括挑战杯、中国国际大学生创新大赛、机械创新大赛等 |
| **通过标准** | ① Top 3 中出现挑战杯或中国国际大学生创新大赛<br>② 理由中提及"有专利"或"有项目可转化"<br>③ 备赛计划针对创新创业类竞赛设计 |

---

**T03 — 设计专业 + 有技能 + 冲校奖**

| 字段 | 内容 |
|---|---|
| **测试类型** | 信息完整用户 |
| **用户输入** | 视觉传达设计专业大二，会 Photoshop、Illustrator、Premiere，有平面设计的作品集。平时每天能找1-2小时做东西，目标是先拿个校奖积累经验。推荐一下适合我的比赛？ |
| **预期路径** | 推荐路径 |
| **预期行为** | 1. 匹配到设计类/视觉类竞赛（如大广赛、计算机设计大赛数媒赛道、未来设计师等）<br>2. 作品集是加分项<br>3. 目标校奖 → 推荐获奖率相对较高的竞赛 |
| **通过标准** | ① Top 3 中包含至少 2 个设计类竞赛<br>② 推荐理由中提及"有作品集"优势<br>③ 目标奖项匹配到"校奖"级别 |

---

**T04 — 经管 + 无编程 + 冲省奖**

| 字段 | 内容 |
|---|---|
| **测试类型** | 信息完整用户 |
| **用户输入** | 工商管理大三，经常做 PPT 和写商业计划书，还会一些市场分析。不会编程。寒假期末比较忙，但开学后每周能抽出 8 小时。想拿省奖，加综测分。 |
| **预期路径** | 推荐路径 |
| **预期行为** | 1. 匹配经管/创新创业类竞赛<br>2. 不会编程 → 不应推荐纯编程竞赛<br>3. 擅长 PPT 和商业计划书 → 推荐创新创业/商科类竞赛 |
| **通过标准** | ① Top 3 中没有蓝桥杯、ACM 等纯编程竞赛<br>② 包含三创赛、市调大赛、商业精英挑战赛等经管类竞赛<br>③ 备赛计划不要求编程技能 |

### C 类：冲国奖用户

> **核心测试目的**：推荐中是否包含适合冲国奖的高难度竞赛，风险是否明确。

---

**T09 — 有实力冲国奖**

| 字段 | 内容 |
|---|---|
| **测试类型** | 冲国奖用户 |
| **用户输入** | 计算机大四，保研边缘人。ACM-ICPC 区域赛银牌，蓝桥杯国二，数学建模国二。寒假和下学期都可以全职备赛，目标是国一或者直接为保研加分。 |
| **预期路径** | 推荐路径 |
| **预期行为** | 1. 因为用户有很强的竞赛基础，推荐高难度国赛<br>2. 推荐竞赛应为 A1/A2 级别的国家级竞赛<br>3. 保研加分信息明确 |
| **通过标准** | ① Top 3 全部为国家级 A1/A2 竞赛<br>② 推荐中提及保研加分情况<br>③ 风险分析客观但不打击积极性 |

### D 类：时间不足用户

> **核心测试目的**：是否推荐短期竞赛，是否对不合理时间要求给出提醒。

---

**T10 — 时间极短**

| 字段 | 内容 |
|---|---|
| **测试类型** | 时间不足用户 |
| **用户输入** | 计算机大二，会 Python。但比赛两周后就截止了，有什么来得及的？ |
| **预期路径** | 推荐路径 |
| **预期行为** | 1. 匹配时间等级为 1（短期）的竞赛<br>2. 推荐个人赛为主的竞赛<br>3. 在风险提醒或备注中说明"时间非常紧张" |
| **通过标准** | ① 推荐竞赛的准备周期为短期（1 个月以内）<br>② 有风险提醒说明时间紧张<br>③ 没有推荐需要 3 个月以上准备的竞赛 |

---

**T11 — 学期中碎片时间**

| 字段 | 内容 |
|---|---|
| **测试类型** | 时间不足用户 |
| **用户输入** | 自动化大三，会 C 语言和单片机。课很多，每周只能周末花 3-4 小时。有没有能长期慢慢准备的比赛？ |
| **预期路径** | 推荐路径 |
| **预期行为** | 1. 匹配时间等级适中的竞赛<br>2. 备赛计划建议"化整为零"的方式 |
| **通过标准** | ① 备赛计划中有应对碎片时间的安排<br>② 没有推荐需要密集准备的竞赛（如电赛 4 天 3 夜） |

### E 类：只有代码基础

> **核心测试目的**：是否会推荐需要 PPT/答辩/文案等其他能力的竞赛而不提示风险。

---

**T12 — 纯编程，无展示能力**

| 字段 | 内容 |
|---|---|
| **测试类型** | 只有代码基础 |
| **用户输入** | 软件工程大二，Java 和 Python 都熟练，LeetCode 刷了 200 题。但是完全不会做 PPT，答辩也很紧张，有没有不需要展示的比赛？ |
| **预期路径** | 推荐路径 |
| **预期行为** | 1. 优先推荐纯编程/算法类竞赛（蓝桥杯、ACM、天梯赛等）<br>2. 如果推荐创新创业类，必须明确说明"需要补足 PPT 和答辩能力"<br>3. 风险分析中提及"展示能力弱" |
| **通过标准** | ① Top 3 以算法/编程赛为主<br>② 如果出现需答辩的竞赛，有风险提示<br>③ 备赛计划不要求制作 PPT 和路演 |

### F 类：只有 PPT / 设计能力

|---|---|

| **预期路径** | 推荐路径 |

### G 类：有现成项目

> **核心测试目的**：是否推荐"适合项目转化"的竞赛，备赛计划是否包含"改造现有项目"。

|---|---|

| **通过标准** | ① Top 3 中适合项目转化的竞赛 ≥ 2 个<br>② 推荐理由中包含"项目转化"<br>③ 备赛计划有"改造现有系统"的任务 |

| **通过标准** | ① 推荐中包含挑战杯（大挑）或节能减排大赛<br>② 理由中有"论文可转化参赛"<br>③ 专业匹配到环境/综合类竞赛 |

| **通过标准** | ① 回答中不含压力语言<br>② 推荐难度不超过"中"<br>③ 有鼓励性的引导语 |

### I 类：专业与比赛不匹配

> **核心测试目的**：是否阻止明显不合理的搭配，或给出清晰的风险提示。

---

**T18 — 文科想参加纯编程竞赛**

| 字段 | 内容 |
|---|---|
| **测试类型** | 专业不匹配 |
| **用户输入** | 我是汉语言文学大三，平时喜欢写文章。我想参加蓝桥杯，行不行？ |
| **预期路径** | 推荐路径 |
| **预期行为** | 1. 匹配评分中专业匹配分很低（甚至 0 分）<br>2. 推荐中说明"专业匹配度低"<br>3. 备选推荐更适合的竞赛（如中华经典诵写讲、外研社·国才杯等） |
| **通过标准** | ① 不直接说"你不适合"<br>② 给出了更匹配的替代推荐<br>③ 没有将蓝桥杯排在 Top 3 的首位 |

---

**T19 — 冷门专业查无匹配**

| 字段 | 内容 |
|---|---|
| **测试类型** | 专业不匹配 |
| **用户输入** | 社会体育指导与管理大二。平时课多，但暑假时间充足。想去试试学科竞赛，有什么比赛是不限制专业的吗？ |
| **预期路径** | 推荐路径 |

| **通过标准** | ① 推荐中包含全专业通用的竞赛（挑战杯、iCAN 等）<br>② 没有武断说"没有适合你的比赛" |

### J 类：不合理要求

> **核心测试目的**：智能体是否友善地指出不合理之处，同时给出建设性方案。

---

**T20 — 一周国奖**

| 字段 | 内容 |
